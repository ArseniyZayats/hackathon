Інтерактивний веб-дашборд для глибокої аналітики, 3D-візуалізації та AI-оцінки "чорних скриньок" (логів `.BIN`) безпілотних літальних апаратів на базі **Ardupilot**. 

Проєкт розроблено в рамках інженерного хакатону **BEST**.

## ✨ Головні можливості

* 📂 **Глибокий парсинг MAVLink:** Вилучення даних GPS, IMU (акселерометр, гіроскоп), ATT (крен, тангаж, рискання), MODE (режими автопілота) та CURR (батарея) прямо з сирих бінарників `.BIN`.
* 📊 **Кінематична аналітика:** Автоматичний розрахунок реальної швидкості, пройденої дистанції, максимальної висоти та прискорень.
* 🗺️ **Просторова траєкторія:** * **2D GPS Карта:** Відображення реального маршруту на світовій карті (WGS-84).
  * **3D Анімація:** Динамічна симуляція польоту з колоруванням траєкторії залежно від швидкості дрона.
* ⚖️ **Режим порівняння:** Одночасне завантаження двох лог-файлів для візуального та математичного порівняння ефективності місій.
* 🤖 **AI-Асистент (Gemini):** Інтегрована LLM для автоматичного формування професійних технічних висновків про агресивність польоту та наявність аномалій.

## 📁 Структура проєкту

* `main.py` — Головний файл веб-інтерфейсу (Streamlit).
* `parser.py` — Модуль низькорівневого зчитування бінарних логів (на базі `pymavlink`).
* `CoreAnalytics/` — Математичне ядро для фільтрації, злиття даних (`merge_asof`) та розрахунку похідних метрик.
* `3DVisual/` — Рушій для перетворення географічних координат у локальні декартові (ENU) та рендерингу 3D-анімацій.
* `requirements.txt` — Список залежностей.

## 🚀 Встановлення та локальний запуск

### 1. Клонування репозиторію
```bash
git clone [https://github.com/ВАШ_ЮЗЕРНЕЙМ/hackathon.git](https://github.com/ВАШ_ЮЗЕРНЕЙМ/hackathon.git)
cd hackathon
```

### 2. Встановлення залежностей
Рекомендується використовувати віртуальне середовище (venv):
```bash
pip install -r requirements.txt
```

### 3. Налаштування AI (Google Gemini)
Для роботи Штучного Інтелекту потрібен безкоштовний API ключ [Google AI Studio](https://aistudio.google.com/).
Створіть папку `.streamlit` та файл `secrets.toml` всередині неї:
```bash
mkdir .streamlit
```
Додайте свій ключ у файл `.streamlit/secrets.toml`:
```toml
GEMINI_API_KEY = "твій_секретний_ключ"
```

### 4. Запуск дашборду
```bash
streamlit run main.py
```
*Після цього застосунок автоматично відкриється у вашому браузері за адресою `http://localhost:8501`.*

## 🕹️ Як користуватися

1. Відкрийте застосунок.
2. У бічній панелі зліва завантажте один лог-файл `.BIN` (наприклад, `00000001.BIN`), щоб отримати детальний звіт.
3. Щоб увімкнути **режим порівняння**, виділіть і завантажте **два** файли одночасно.
4. Досліджуйте графіки напруги, лог режимів польоту та 3D-модель (використовуйте кнопку Play під 3D-графіком для запуску анімації).
5. Натисніть кнопку **"Згенерувати автоматичний звіт"** внизу сторінки для отримання AI-аналітики.
